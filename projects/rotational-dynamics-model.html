Helicopter Pitch Control: PID Tuning & Response Comparison

← Back to Home

Objective

Model a second-order pitch and roll dynamic system representative of helicopter body motion. Implement a control system using Simulink to stabilize angular response under step commands and external disturbance torques.

System Overview

  • Model type: Second-order linear dynamics (θ'' = M/I)
  • Inputs: Step input (desired angle), Sine wave disturbance (wind torque)
  • Controller: PID (Proportional, Integral, Derivative)
  • Output: Angular position (θ)

Control Architecture

The closed-loop control system used Simulink to implement a PID controller. Error was computed as the difference between desired angle and actual angle, and fed into the PID. The controller output torque combined with a disturbance signal before entering the plant block.

Simulation Results

Three sets of PID gains were tested to evaluate response:

Test Set P I D Notes
A 2 0.2 0.4 Stable, slight overshoot, settles within ±1°
B 3 0.8 0.5 Faster rise time, some overshoot, still stable
C 1 0 0 Under-damped, significant steady-state error (~20°)

Key Takeaways

  • Proportional gain increases responsiveness but risks overshoot
  • Integral gain removes steady-state error but can slow response
  • Derivative gain damps oscillation and improves settling time
  • Without integral action, the system was unable to reject constant bias from disturbances
